SPRITE SHEET GUIDE — Stick Figure Combat
=========================================

Create sprite sheets in Pivot Animator and export as PNG sequences,
then combine each animation into a single-row sprite sheet.

FORMAT
------
- Each animation = one PNG file
- Frames laid out LEFT to RIGHT in a single horizontal row
- Recommended frame size: 64 x 128 pixels
- Transparent background (PNG alpha)
- Stick figure centered horizontally, feet at the bottom edge

EXPECTED FILES
--------------
  idle.png          1 frame    Standing still
  walk.png          8 frames   Walk cycle (full stride loop)
  light_attack.png  6 frames   Quick jab / light punch
  heavy_attack.png  8 frames   Strong swing / heavy hit
  block.png         1 frame    Defensive stance

EXAMPLE DIMENSIONS
------------------
  idle.png         →   64 x 128  (1 frame  × 64w)
  walk.png         →  512 x 128  (8 frames × 64w)
  light_attack.png →  384 x 128  (6 frames × 64w)
  heavy_attack.png →  512 x 128  (8 frames × 64w)
  block.png        →   64 x 128  (1 frame  × 64w)

PIVOT ANIMATOR WORKFLOW
-----------------------
1. Create your stick figure with joints in Pivot's Figure Builder
2. Animate each action (walk cycle, attacks, etc.) as separate projects
3. File → Export Animation → Separate Images (PNG)
4. Use an image editor or tool to stitch the frames into a single row
   (e.g., ImageMagick: convert frame*.png +append walk.png)

MODIFYING FRAME COUNTS / SIZES
-------------------------------
If your sprite sheets use different dimensions or frame counts,
update the SPRITE_DEFS object in stick-combat.html to match:

  const SPRITE_DEFS = {
    walk: { src: 'sprites/walk.png', frames: 8, frameW: 64, frameH: 128, fps: 12 },
    ...
  };
